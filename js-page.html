Apprenez à programmer avec JavaScript / Créez des pages web interactives avec JavaScript

JavaScript est un langage de programmation principalement employé dans les pages web interactives et à ce titre est une partie essentielle des applications web. JavaScript est parfois considéré comme l'une des technologies cœur du World Wide Web.

Qu'est-ce que j'ai appris grâce à ces formations ?

Ces formations sont la continuité des formations HTML CSS et Bootstrap. Ainsi ces formations m'ont fait découvrir un nouvel aspect du développement web et m'ont permis de m'améliorer techniquement.

    Voici quelques exemples :
  1. La syntaxe
  2. L'utilisation du DOM
  3. La modification de la structure d'une page après son chargement
  4. La modification du style d'une page de manière dynamique
  5. La gestion des événements
  6. Mettre en place des animations
  7. L'intéraction avec un serveur web

Contenu des formations

La formation "Apprenez à programmer avec JavaScript" est constitué de trois parties.

  • La première est d'utiliser les données et les types de données dans JavaScript
  • La seconde est de gérer la logique d'un programme en JavaScript (conditions, boucles et erreurs)
  • La troisième est d'écrire du code propre et facile à maintenir à l'aide de méthodes en JavaScript


    • La formation "Créez des pages web interactives avec JavaScript" est constitué également de trois parties.
  • La première est la découverte du DOM
  • La seconde la gestion des événements et l'animation
  • La troisième l'intéraction avec un serveur web

    • Pourquoi ces formations ?

    Afin de découvrir et prolonger mes connaissances du langage JavaScript j'ai effectué deux formations sur la plateforme openclassrooms. La première m'a permis de découvrir le language JS dans le but de maitriser la syntaxe. La seconde formation a été la continuité logique afin d'utiliser le language JS pour développer une page web.
