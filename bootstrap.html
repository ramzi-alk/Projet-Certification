Le framework CSS

Créez des sites rapides et réactifs

Bootstrap est un framework open source développé par l'équipe du réseau social Twitter, qui fournit aux développeurs des outils pour créer un site facilement. Ce framework est pensé pour développer des sites avec un design responsive, qui s'adapte à tout type d'écran (smartphone, tablette...).

Qu'est-ce que j'ai appris grâce à cette formation ?

La formation bootstrap m'a permis de gagner énormément de temps lors de la conception de certains projets que j'ai dû réaliser. Le responsive étant géré automatiquement le développement front-end a donc été plus rapide.

Contenu de la formation

  1. Identifier les avantages de Bootstrap.
  2. Créer des mises en page responsives.
  3. Utiliser les composants d'interface utilisateur de Bootstrap pour afficher le contenu.
  4. Ajouter de l'interactivité à un site web.
  5. Créer des fonctionnalités et des thèmes personnalisés à l'aide de Bootstrap.

Pourquoi cette formation ?

J'ai choisi cette formation afin de découvrir le fonctionnement d'un framework CSS et ainsi coder plus facilement la partie front-end d'un site web. De plus Bootstrap permet de retrouver une structure HTML standardisée et cela permettra à un autre développeur utilisant Bootstrap de mieux comprendre le code qu'il a devant lui, ce qui favorise le travail en groupe. Ce site utilise notamment bootstrap.
